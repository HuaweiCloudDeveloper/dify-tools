## 仓库简介

本仓库提供了Dify中连接华为云商店产品的一些插件的源码以及一些相关的Workflow和Demo，供Dify用户和华为云用户参考借鉴。

## 前置条件

- 华为云账号
- Dify环境（可以通过华为云商店一键部署社区版）
- 基本的Linux环境使用经验
